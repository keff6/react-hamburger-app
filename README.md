# react-hamburger-app

Application for the Udemy course "React 16 - The complete guide" by Maximilian Schwarzmuller.

## Key Topics

- ReactJS
- Redux
- react-router
- Error Handling
- axios

## Installation

* git clone <repository-url> this repository
* change into the new directory
* npm install

## Run

* npm start
* Visit your app at http://localhost:3000.
